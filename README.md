---

## 📌 Overview

DekNek TaskFlow is a full-stack task management web application that allows users to securely manage their daily tasks.

It includes user authentication, persistent data storage, and a fully deployed architecture using modern web technologies.

---

## ✨ Features

- 🔐 User Authentication (Signup & Login)
- 📝 Create and Delete Tasks
- 👤 User-specific task management
- 💾 Persistent data with MongoDB
- 🌐 Fully deployed (Frontend + Backend)
- ⚡ Fast and responsive UI

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas

### Deployment
- Frontend: Vercel
- Backend: Render

---

## 🧪 How It Works

1. User signs up or logs in
2. JWT authentication verifies user identity
3. Tasks are stored in MongoDB Atlas
4. Each user sees only their own tasks
5. Data persists across sessions and devices
